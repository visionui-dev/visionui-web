Everything you need to build high-performance desktop applications with VisionUI Framework.

bolt Quick Start Guide

1

Install via pip

Add VisionUI to your Python environment.

pip install visionui
2

Import the framework

Set up your project and import the components you need.

from visionui import VUIApp, GlassWidget, VUIButton
3

Create your first window

Initialize your application and deploy it in seconds.

app = VUIApp(theme='glass')
window = app.create_window("My App")
app.run()
